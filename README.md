> Wrapper of dangerjs in ts

## Usage

Install:

```sh
yarn add danger-plugin-wrapper-ts --dev
```

At a glance:

```js
// dangerfile.js
import wrapperTs from 'danger-plugin-wrapper-ts'

wrapperTs()
```
